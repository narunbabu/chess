# Live Move Review Report

Added live Best/Review tracking for games:

- Casual Best usage is counted and Best suggestions clear after a legal move.
- Review can be enabled during any mode and records the user's move rank against the top five engine candidates.
- Logged-in game history records now persist review report JSON, summary JSON, Best usage count, and Review usage state.
- Game end and Game Review screens can open a report modal when live review data exists.
