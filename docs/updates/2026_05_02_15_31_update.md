# Mobile Play Area Redesign

## Summary

- Redesigned active play on mobile so the chessboard remains the primary surface.
- Added a compact landscape side rail for player bars, tool controls, Moves, CCT, Companion, Help, Undo, and Resign.
- Changed mobile move history into a fixed drawer so move text no longer resizes or flickers the board.
- Updated the guided tour target lookup to prefer visible elements, which supports responsive duplicate controls.
- Added tour targeting hooks for CCT, Best moves, and companion controls.
- Added Playwright coverage for mobile portrait and landscape play layouts.

## Verification

- `pnpm build` passed.
- `pnpm test:e2e -- tests/e2e/mobile-play-layout.spec.js --project=chromium` passed across the configured Playwright projects.
