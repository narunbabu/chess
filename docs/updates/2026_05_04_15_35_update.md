# Backend Training Drill Foundation

Date: 2026-05-04 15:35

## Summary

Implemented the backend-managed training drill foundation for the Lessons and Training Drills overhaul.

## Changes

- Added training drill catalog tables:
  - `training_drill_sets`
  - `training_drills`
  - `user_training_drill_progress`
  - `user_training_drill_attempts`
- Seeded foundational drill content across:
  - Newcomer
  - Beginner
  - Improving Beginner
  - Club Player
  - Advanced
  - Competitive
- Added backend models for drill sets, drills, user progress, and attempts.
- Added `TrainingDrillController` with endpoints for:
  - listing drills with tier lock metadata
  - recommended drills
  - drill detail
  - attempt submission and progress updates
- Registered routes under both:
  - `/api/training/...`
  - `/api/v1/training/...`
- Updated the frontend Training Hub and exercise view to prefer the backend drill catalog, with local fallback before migrations are available.
- Added focused feature tests for drill listing, tier access, attempts, and progress.
