## Key Decisions

- Existing `chapter-01.json` through `chapter-06.json` should be treated as prototype content, not the foundation for the serious curriculum.
- Plain Markdown should not be the canonical format for Chess99 ebooks.
- The canonical source should be structured JSON book packages with Markdown only inside prose fields.
- Authoritative chess diagrams must be generated deterministically from FEN/arrows/highlights, not from general image generation.
- AI image generation should be used for covers and non-authoritative explanatory art.
- Every chapter needs schema validation, chess validation, diagram generation, text-to-image consistency review, pedagogy review, and human chess signoff.
- The curriculum should be built as 12 books across Free, Silver, and Gold tiers.
