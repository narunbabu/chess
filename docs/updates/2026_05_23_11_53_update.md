# Best Move Arrows Fix

## Summary

Fixed Best move arrows disappearing on mobile by stabilizing the Best results callback and allowing the main Best button to toggle the overlay off cleanly.

## Changes

- Kept `GameContainer`'s `onBestMovesReady` bridge stable with a ref so `CCTPanel` does not restart and clear the Stockfish effect after parent rerenders.
- Changed external Best requests to toggle the CCTPanel Best state instead of always forcing it on.
- Reduced review arrow opacity and stroke scale so dotted review arrows are visually thinner than solid Best arrows.
- Added a mobile Playwright regression that fakes Stockfish, verifies Best arrows plus the inline strip, then verifies a second Best tap clears both.
- Updated stale mobile layout assertions to match the current promoted Pause/Resume mobile control.
