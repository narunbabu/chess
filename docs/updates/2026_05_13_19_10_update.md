# Sub-1300 Stockfish Wrapper

## Summary

Added a Stockfish-backed beginner move policy for 200-1300 Elo targets, snapped to 50-point intervals.

## Changes

- Added `chess-frontend/src/utils/sub1300Stockfish.js`.
- Routed `makeComputerMove` through the sub-1300 policy when the target Elo is 1300 or lower.
- Extended Stockfish MultiPV search calls to support profile-specific depth and timeout options.
- Added unit tests for profile snapping, search config bounds, illegal candidate filtering, legal fallback moves, and low-Elo blunder selection.
- Added seeded synthetic-vs-synthetic simulation coverage for 200, 600, 900, 1000, and 1300 Elo pairings. The test verifies every selected move is legal and stronger profiles choose lower average centipawn-loss moves over paired-color simulations.
