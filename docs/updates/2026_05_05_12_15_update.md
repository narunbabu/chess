# Entitlement Foundation for School and Tier Access

## Summary

- Added a backend entitlement service as the central source for Free, Silver, Gold, and future school-pass capability checks.
- Exposed authenticated entitlement summaries through `/api/entitlements/me` and `/api/v1/entitlements/me`.
- Updated online game quota checks so Silver users receive unlimited online games, matching the planned tier promise.
- Updated training drill access to use capability checks instead of direct tier checks.
- Fixed the subscription grace-period check so entitlement checks do not mutate `subscription_expires_at` in memory.
- Added frontend entitlement context and a reusable feature gate component.
- Added Android entitlement API and repository wiring so mobile screens can use the same capability map.
